# Pipelight Knowledge Base

This directory contains shared project knowledge — architecture decisions, environment setup, and design vision. It is committed to Git so that all machines and all Claude Code sessions share the same context.

## Structure

- `vision.md` — Project vision and design philosophy
- `architecture.md` — Architecture decisions and roadmap
- `dev-environment.md` — Multi-machine development environment setup
- `decisions.md` — Key technical decisions log
